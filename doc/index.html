This documentation was made for those wishing to improve their athletic training program.

Purpose

PowerBuilder was first created with the purpose of programming training routines to prevent muscle imbalances and general overtraining.

Also, with the intention of it's use to be for general athletic performance.

Thank you for reading this documentation.

Features

  • Body weight, water, fat, muscle mass and metabolic basal rate programming and track.
  • Pysique Measurements, goals and progress programming and tracking
  • Westside BB Style programming and tracking.
  • PowerLifting personal records, goal, progress, programming and tracking.
  • Exercises categorized by muscle group.
  • Nutrition
  • Supplementation categorized by goal.
